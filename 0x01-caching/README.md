# 0x01. Caching

## Description
This project covers the basics of caching, including different caching strategies and how to implement them in Python.

## Learning Objectives
- Understand what caching is and its benefits
- Learn about different caching algorithms
- Implement caching in Python using various libraries

## Requirements
- Python 3.x
- pip

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```python
from cache import Cache

cache = Cache()
cache.set('key', 'value')
print(cache.get('key'))
```
